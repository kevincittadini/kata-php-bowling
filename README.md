# PHP Bowling Game Kata

This Kata is inspired by [Uncle Bob's Kata](http://butunclebob.com/ArticleS.UncleBob.TheBowlingGameKata).


## Requirements

* PHP >= 7.0.0
* Composer


## Install & Run tests

After cloning this repository, run the followin commands:

```
$ composer install
$ ./vendor/bin/phpunit tests --colors=always
```
